
1. Run gamos: 
	gamos miniSPECT_gamma_VISUAL.in
2. This will give you a visualisation file: g4_00.wrl
	(If you run gamos again, you will get a new file with the name: g4_01.wrl)
3. You can open this with the view3dscene program, e.g.:
	view3dscene g4_00.wrl
4. Use the mouse to change the perspective of the visualisation.
